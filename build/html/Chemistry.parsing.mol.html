Chemistry.parsing.mol.molv2000 module¶

class Chemistry.parsing.mol.molv2000.MolV2000(molfile, from_string=False)[source]¶

Bases: object

Represents a molfile that follows the v2000 standard as a data structure in memory (instead of a text file/string)

Lines 1-3 represent the header
1: Title
2: Software produced by
3: Comment line
Lines 4+ are the ctab block. It can be split into a few sections:
Counts line
Atom block
Bond block
Atom list block
Stext (structural text descriptor)
Properties block

Blank numerical values should be represented as 0. Spaces are significant and can indicate any of a number of things, such as:

  • absense of an entry
  • Empty character positions
  • Spaces between entries (single unless noted otherwise)
